Exercice 1 : Calcul littéral – Développement et factorisation ★☆☆

Énoncé :
1) Développer et réduire : A = (2x - 3)(x + 5) - (x - 2)²
2) Factoriser : B = 9x² - 25 + (3x - 5)(x + 1)
✅ Correction :
1) A = (2x - 3)(x + 5) - (x - 2)²
   = 2x² + 10x - 3x - 15 - (x² - 4x + 4)
   = 2x² + 7x - 15 - x² + 4x - 4
   = x² + 11x - 19
2) B = 9x² - 25 + (3x - 5)(x + 1)
   = (3x - 5)(3x + 5) + (3x - 5)(x + 1)
   = (3x - 5)[(3x + 5) + (x + 1)]
   = (3x - 5)(4x + 6)
   = 2(3x - 5)(2x + 3)

Exercice 2 : Équations et inéquations ★☆☆

Énoncé :
1) Résoudre dans ℝ : 3(x - 2) - 2(1 - x) = 5x + 1
2) Résoudre dans ℝ : (2x + 1)/(x - 3) ≤ 0
3) Résoudre dans ℝ : x² - 5x + 6 = 0
✅ Correction :
1) 3(x - 2) - 2(1 - x) = 5x + 1
3x - 6 - 2 + 2x = 5x + 1
5x - 8 = 5x + 1
-8 = 1 → Impossible. S = ∅
2) (2x + 1)/(x - 3) ≤ 0
Signe de 2x + 1 : négatif sur ]-∞; -1/2], positif sur [-1/2; +∞[
Signe de x - 3 : négatif sur ]-∞; 3[, positif sur ]3; +∞[
Tableau de signes : solution S = [-1/2; 3[
3) x² - 5x + 6 = 0
Δ = 25 - 24 = 1
x₁ = (5 - 1)/2 = 2, x₂ = (5 + 1)/2 = 3
S = {2; 3}

Exercice 3 : Fonctions affines et linéaires ★★☆

Énoncé :
Soit f la fonction affine définie par f(x) = -2x + 4.
1) Déterminer l'image de 3 par f.
2) Déterminer l'antécédent de 0 par f.
3) Tracer la droite représentative de f dans un repère.
4) Déterminer algébriquement les coordonnées du point d'intersection de la droite avec l'axe des abscisses.
✅ Correction :
1) f(3) = -2 × 3 + 4 = -6 + 4 = -2. L'image de 3 est -2.
2) On cherche x tel que f(x) = 0 : -2x + 4 = 0 → x = 2. L'antécédent de 0 est 2.
3) La droite passe par A(0; 4) et B(2; 0). Coefficient directeur = -2.
4) Intersection avec l'axe des abscisses : y = 0
-2x + 4 = 0 → x = 2. Le point d'intersection est (2; 0).

Exercice 4 : Statistiques – Moyenne et écart-type ★★☆

Énoncé :
Les notes obtenues par 10 élèves à un contrôle sont : 8, 12, 15, 10, 7, 14, 11, 9, 13, 16.
1) Calculer la moyenne de cette série.
2) Calculer l'écart-type (arrondi au dixième).
3) Déterminer le pourcentage d'élèves ayant une note supérieure ou égale à la moyenne.
✅ Correction :
1) Moyenne = (8+12+15+10+7+14+11+9+13+16)/10 = 115/10 = 11,5
2) Variance = [(8-11,5)² + (12-11,5)² + ... + (16-11,5)²]/10
= [12,25 + 0,25 + 12,25 + 2,25 + 20,25 + 6,25 + 0,25 + 6,25 + 2,25 + 20,25]/10
= 82,5/10 = 8,25
Écart-type = √8,25 ≈ 2,9
3) Notes ≥ 11,5 : 12, 15, 14, 13, 16 → 5 élèves sur 10
Pourcentage = 50%

Exercice 5 : Géométrie dans le plan – Vecteurs ★★☆

Énoncé :
Dans un repère orthonormé, on donne les points A(2; 3), B(5; 1) et C(-1; 4).
1) Calculer les coordonnées du vecteur AB.
2) Calculer la longueur AB.
3) Déterminer les coordonnées du point D tel que ABCD soit un parallélogramme.
✅ Correction :
1) AB = (xB - xA; yB - yA) = (5-2; 1-3) = (3; -2)
2) AB = √(3² + (-2)²) = √(9 + 4) = √13
3) ABCD parallélogramme ⇔ AB = DC
(3; -2) = (-1 - xD; 4 - yD)
-1 - xD = 3 → xD = -4
4 - yD = -2 → yD = 6
D(-4; 6)

Exercice 6 : Proportionnalité et pourcentages ★☆☆

Énoncé :
Un article coûte 80 €. Son prix augmente de 15%, puis diminue de 20% lors des soldes.
1) Calculer le prix après la première augmentation.
2) Calculer le prix final après la baisse.
3) Quel est le pourcentage d'évolution global entre le prix initial et le prix final ?
✅ Correction :
1) Prix après augmentation = 80 × 1,15 = 92 €
2) Prix final = 92 × 0,80 = 73,60 €
3) Coefficient global = 1,15 × 0,80 = 0,92
Évolution = (0,92 - 1) × 100 = -8%
Le prix a baissé de 8% au total.

Exercice 7 : Fonction de référence – Carré et inverse ★★☆

Énoncé :
On considère les fonctions f(x) = x² et g(x) = 1/x définies sur ℝ* (pour g).
1) Compléter le tableau de variations de f sur [-3; 3].
2) Résoudre f(x) = g(x) dans ℝ*.
3) Déterminer le signe de g(x) - f(x) sur ]0; +∞[.
✅ Correction :
1) f(x) = x² : décroissante sur [-3; 0], croissante sur [0; 3], minimum en 0 (f(0)=0)
2) x² = 1/x → x³ = 1 → x = 1. S = {1}
3) g(x) - f(x) = 1/x - x² = (1 - x³)/x
Sur ]0; 1[ : 1 - x³ > 0 et x > 0 → positif
Sur ]1; +∞[ : 1 - x³ < 0 et x > 0 → négatif
En x = 1 : nul

Exercice 8 : Algorithmique – Boucle et condition ★★☆

Énoncé :
On considère l'algorithme suivant :

  n ← 0
  Tant que n² < 100 faire
    n ← n + 1
  Fin Tant que
  Afficher n

1) Quelle valeur affiche cet algorithme ? Justifier.
2) Modifier l'algorithme pour qu'il affiche le plus petit entier n tel que 2ⁿ > 1000.
✅ Correction :
1) L'algorithme cherche le plus petit entier n tel que n² ≥ 100.
9² = 81 < 100, 10² = 100 ≥ 100.
L'algorithme affiche 10.
2)   n ← 0
  Tant que 2ⁿ ≤ 1000 faire
    n ← n + 1
  Fin Tant que
  Afficher n
  (Réponse : n = 10 car 2⁹ = 512 et 2¹⁰ = 1024)

Exercice 9 : Trigonométrie – Cosinus et sinus ★★☆

Énoncé :
Dans un triangle rectangle ABC, rectangle en A, on a AB = 6 cm et AC = 8 cm.
1) Calculer BC.
2) Calculer cos(B̂) et sin(B̂).
3) En déduire la mesure de l'angle B̂ (arrondie au degré).
✅ Correction :
1) BC² = AB² + AC² = 36 + 64 = 100 → BC = 10 cm
2) cos(B̂) = AB/BC = 6/10 = 0,6
sin(B̂) = AC/BC = 8/10 = 0,8
3) B̂ = arccos(0,6) ≈ 53° (ou arcsin(0,8) ≈ 53°)

Exercice 10 : Problème ouvert – Optimisation ★★★

Énoncé :
Un jardinier dispose d'un grillage de 20 mètres pour délimiter un terrain rectangulaire contre un mur (il n'a pas besoin de grillage le long du mur).
1) Exprimer l'aire du terrain en fonction de la longueur x du côté perpendiculaire au mur.
2) Pour quelle valeur de x l'aire est-elle maximale ?
3) Quelle est cette aire maximale ?
✅ Correction :
1) Si x est la largeur (côtés perpendiculaires au mur), la longueur est 20 - 2x.
Aire A(x) = x(20 - 2x) = 20x - 2x², avec 0 < x < 10.
2) A(x) = -2x² + 20x = -2(x² - 10x) = -2[(x - 5)² - 25] = -2(x - 5)² + 50
Maximum atteint en x = 5 m (sommet de la parabole).
3) Aire maximale = 50 m². Le terrain est un rectangle de 5 m × 10 m.
